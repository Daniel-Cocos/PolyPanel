# PolyPannel
PolyPannel backend runs locally with `docker compose up --build`. The frontend is deployed separately from `frontend/` with `npm run deploy`, which publishes only `frontend/dist` to the `gh-pages` branch for GitHub Pages hosting. The landing page includes a responsive header that collapses to a hamburger menu on mobile.
